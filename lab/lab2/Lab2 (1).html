# 4.1 Create a scatter plot of longitude and latitude, colored by waterfront
fig, ax = plt.subplots(figsize=(10,8))
ax.scatter(df_subset.loc[df_subset['waterfront'] == 0, 'long'], df_subset.loc[df_subset['waterfront'] == 0, 'lat'], color='pink', alpha=0.5, label='Non-waterfront')
ax.scatter(df_subset.loc[df_subset['waterfront'] == 1, 'long'], df_subset.loc[df_subset['waterfront'] == 1, 'lat'], color='blue', alpha=0.5, label='Waterfront')
ax.set_xlabel('Longitude')
ax.set_ylabel('Latitude')
ax.set_title('longitude/latitude of the listings.')
ax.legend(loc='upper right')
plt.show()

4.2¶

As seen on the map, it shows the shape of Seattle.The waterfront properties are near lakes and sea shore.This makes logic, as waterfront properties are typically more desirable and thus cost more. The non-waterfront properties are more equitably distributed across the region.
